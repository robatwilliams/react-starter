# React Starter

## Features
Framework & ecosystem:
* React
* React Router

Code:
* TypeScript

Styling:
* CSS Modules

Build:
* Webpack
* Source maps (TypeScript, styles)

Development:
* Webpack Dev Server
* Tool configuration
  * Visual Studio Code (settings)

## Non-features
* CSS in JS
* Offline support
* Server-side rendering

## Principles
1. Simplicity
1. No hidden magic
1. Explain the "why" when it isn't clear
1. No obscure dependencies
1. No unstable features/tools
